# AOZ-MPIT. Интерактивный WEB-сайт: визитка Новороссии
Интерактивный **сайт-визитка** субъектов Новороссии для привлечения туристов, осведомления о культуре каждого субъекта Новороссии

## Здесь представлены 4 субъекта РФ
- Луганская Народная Республика
- Донецка Народная Республика
- Запорожская область
- Херсонская область

## Использованные технологии
- Python
  - Flask
- WEB-программирование
  - HTML
  - CSS
  - JavaScript
- WEB-дизайн
  - Figma

## Как уствновить?
1. Установите библиотеки
`pip install -r requirements.txt`
2. Запустите файл `main.py`
3. Перейдите по локальному адресу http://localhost

Проект сделан в рамках Всероссийского конкурса "Моя профессия - ИТ" (2025 г.)
